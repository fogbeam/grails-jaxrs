Package: org.grails.plugins.jaxrs.provider

[Java] Class XMLWriter

        • org.grails.plugins.jaxrs.provider.XMLWriter

        • Provider for Grails' grails.converters.XML converter class. Message body writer that supports grails.converters.XML response entities. For example, to create an XML representation from a list of Grails domain objects: 

           @Path('/notes')
 @Produces('text/xml') // or 'application/xml'
 class NotesResource {

      @GET
      XML findNotes() {
          Note.findAll() as XML
      }

 }
          Alternatively, one could write 
           @Path('/notes')
 @Produces('text/xml') // or 'application/xml'
 class NotesResource {

      @GET
      Response findNotes() {
          Response.ok(Note.findAll() as XML).build()
      }

 }
