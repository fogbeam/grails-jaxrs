[Java] Class JSONReader

        • org.grails.plugins.jaxrs.provider.JSONReader
        • All Implemented Interfaces and Traits:
          grails.core.support.GrailsApplicationAware

          @javax.ws.rs.ext.Provider
@javax.ws.rs.Consumes({"text/x-json", "application/json"})
public class JSONReader
extends MessageBodyReaderSupport

          A message body reader that converts a JSON entity stream to a map than can be used to construct Grails domain objects. Any JAX-RS resource method that defines a java.util.Map as parameter and consumes either text/x-json or application/json will be passed a map created from a JSON request entity: 

           @Path('/notes')
 @Produces('text/x-json')
 class NotesResource {

      @POST
      @Consumes('text/x-json')
      Response addNote(Map properties) {
          // create ne Note domain object
          def note = new Note(properties).save()
      }

 }
