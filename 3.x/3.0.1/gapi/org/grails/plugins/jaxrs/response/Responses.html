Package: org.grails.plugins.jaxrs.response

[Groovy] Class Responses

    • org.grails.plugins.jaxrs.response.Responses

    • Factory methods for JAX-RS response objects. This class is used by JAX-RS resource implementations to create XML responses from Grails domain objects. Resources generated with the command 

       grails generate-resources <domain-object-class>
      make use of these factory methods.
      Authors:
      Martin Krasser
